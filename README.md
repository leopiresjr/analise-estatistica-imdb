# Análise Exploratória de Dados - IMDb Top 1000

Projeto desenvolvido como trabalho final da disciplina de Estatística no curso de Análise de Dados.

## Objetivo
Realizar uma análise exploratória completa de uma base de dados real contendo os 1000 filmes mais bem avaliados do IMDb.

## Principais análises realizadas
- Estatísticas descritivas
- Histogramas
- Análise de frequências
- Correlação entre variáveis
- Scatter plot
- Boxplot
- Perguntas analíticas com filtragens e gráficos
- Limpeza e avaliação da base

## Tecnologias utilizadas
- Python
- Pandas
- Matplotlib
- Google Colab

## Arquivos
- `estatistica_final.ipynb` → notebook completo
- `imdb_top_1000.csv` → base de dados utilizada
